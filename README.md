# MyRecipe Web Application

## Introductory Information

Welcome to **MyRecipe**, a web application built using the Python-based Django web framework development server. In the MyRecipe web application, users have the ability to create, read, and modify recipes, and search for recipes by ingredients. The application also includes an admin panel for data handling and visualization.

The MyRecipe application is a full-stack web application deployed and hosted using Heroku, with a Postgres database at the backend, HTML, and CSS-based rendered pages at the frontend and, as previously mentioned, Python-based Django as the web application framework.

MyRecipe is dynamic and multi-user, allowing users to sign up and create their own content. In addition to what users can create and perform within the application, MyRecipe also has statistical dashboards, implementing data analytics and visualization.

## Summary of MyRecipe Features

- Allows for user authentication, login, and logout.
- Lets users search for recipes according to ingredients.
- Dynamically calculates a difficulty level rating for each recipe.
- Receives user input and handle errors appropriately.
- Displays additional details on each recipe if the user asks for it.
- Adds user recipes to an SQLite database for storage.
- Includes a Django Admin dashboard for working with database entries.
- Shows statistics and visualizations based on trends and data analysis.
