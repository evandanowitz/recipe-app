BiteBase

A dynamic, user-friendly recipe management web applicationbuilt using the Django web framework.


The development of BiteBase created a fully functional web app using Python-based Django to best demonstrate my understanding of the fundamentals of the Django web framework and my web application development skills using the framework professionally. The app is designed to create, read, modify, search for and delete recipes. It includes an admin panel for data handling and visualization, and implements best practices inweb development, user authentication, and data management.


Key Features

  • User Authentication: Sign up, log in, and manage personal recipes.
  • Recipe Management: Create, edit, and delete recipes with structured details.
  • Search: Find and filter recipes with user search criteria.
  • Difficulty Rating System: Recipes are automatically categorized by difficulty.
  • Data Visualization: View trends and statistics through dynamic chart generation.
  • Admin Panel: Securely manage recipes and users via Django's built-in admin.
  • Deployed with Heroku: The application is live and accessible from anywhere.
